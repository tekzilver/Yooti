Yooti

A modern, sleek, and safe YouTube video downloader powered by Tekzilver.

Yooti Features

    High Compatibility: Downloads MP4 files optimized for Windows Media Player (H.264/AAC).
    Modern UI: Stylish gradient design built with Poppins font.
    Privacy First: No data collection. Runs locally on your machine.
    Easy to Use: Simple "Paste & Download" workflow.

Getting Started

Prerequisites

    Node.js: Download and install Node.js.
    FFmpeg: Download ffmpeg.exe (Static Build) from Gyan.dev.
        Extract the zip, find bin/ffmpeg.exe, and copy it into the project root folder.

Installation

    Clone this repository or download the source code.
    Open a terminal in the project folder.
    Install dependencies:

    npm install

    Build the application:

    npm run build

    The installer will be generated in the dist folder.
